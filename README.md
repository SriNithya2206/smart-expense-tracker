# 💰 Smart Expense Tracker

## 📌 Project Description
This project is a simple expense tracking system built using Python. It allows users to record and manage their daily expenses by category through a command-line interface. The system helps users analyze their spending patterns and monitor total expenses.

---

## 🚀 Features
- Add expenses by category (Food, Travel, Shopping, etc.)
- View all recorded expenses
- Calculate total spending
- Identify highest spending category
- Simple and interactive menu-driven interface

---

## 🛠️ Tech Stack
- Python

---

## ▶️ How to Run the Project

### 1. Save the file
expense_tracker.py

### 2. Run the program
python expense_tracker.py

---

## 📊 Example Input
- Food: ₹200  
- Travel: ₹500  

👉 Output:
- Total Spending: ₹700  
- Highest Spending Category: Travel
